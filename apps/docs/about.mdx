## Who we are

**All Things Linux (ATL)** is a volunteer-driven **501(c)(3)** non-profit. We help people explore, master, and shape Linux through **open education**, **community-built tools**, and a **welcoming social hub**—notably a large Discord community and open-source projects maintained with and for contributors.

**Mantra:** _Connection. Education. Empowerment._

This Mintlify site documents **technical workflows** for our **monorepo** (for example how we author and publish these docs). It is not a full copy of our organizational handbook. For mission, governance, brand, and policies, see the resources below.

## This documentation site

The pages under **Guides** explain how to work on **this repo’s** docs (Mintlify, MDX, CI). The **API reference** tab uses a **sample OpenAPI** specification for the Mintlify playground—it does **not** describe a single production ATL HTTP API unless we publish a dedicated product spec.
