## Overview

**atl.chat** is All Things Linux‚Äôs **unified chat infrastructure**: **IRC** (UnrealIRCd + Atheme services), **XMPP** (Prosody), web apps and clients (Next.js, The Lounge, ObsidianIRC, Fluux Messenger), and a **Python bridge** that connects **Discord ‚Üî IRC ‚Üî XMPP** with multi-presence and format conversion.

The stack is designed as a **Docker Compose** monorepo with a **`just`** task runner: `just init`, `just dev` (localhost + Dozzle), `just prod`, plus per-service recipes for IRC, XMPP, the bridge, and web clients.
