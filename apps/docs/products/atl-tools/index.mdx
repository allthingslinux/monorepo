## Overview

**[atl.tools](https://atl.tools)** is a **curated directory** of **free, self-hosted** services run by All Things Linux: browse, search, and filter tools with no third-party data sharing. The web app is **Next.js** (App Router), **Tailwind v4**, deployed to **Cloudflare Workers** (Alchemy + OpenNext), with service definitions in a workspace **`manifest`** package.

## Examples of hosted services

| Service      | URL                   | Role                   |
| ------------ | --------------------- | ---------------------- |
| PrivateBin   | `paste.atl.tools`     | Encrypted paste        |
| SearXNG      | `search.atl.tools`    | Metasearch             |
| CyberChef    | `cyberchef.atl.tools` | Encoding / analysis    |
| ConvertX     | `convert.atl.tools`   | File conversion        |
| IT-Tools     | `it.atl.tools`        | Dev/sysadmin utilities |
| JSON Crack   | `json.atl.tools`      | JSON/YAML graphs       |
| Stirling PDF | `pdf.atl.tools`       | PDF tooling            |
| hckrnws      | `hn.atl.tools`        | Hacker News frontend   |

New entries are added in the manifest source in the repo.
