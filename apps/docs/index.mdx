## Overview

Welcome. **[All Things Linux](https://allthingslinux.org) (ATL)** is a volunteer-driven **501(c)(3)** non-profit Linux community—open education, community-built tools, and a welcoming hub for contributors and learners. This site documents **how we build and publish docs** for the **[allthingslinux monorepo](https://github.com/allthingslinux)**: Mintlify, MDX, CI checks, and interactive **sample** OpenAPI pages for the playground.
