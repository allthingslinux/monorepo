## Overview

**tux** is All Things Linux’s **Discord bot**: moderation with case management, **leveling** / XP, **snippets**, utilities, fun commands, and a **plugin** architecture. It targets **Python 3.13+**, **`discord.py`**, **`uv`**, **PostgreSQL** + **SQLModel**, **`ruff`**, **`pytest`**, optional **Valkey** caching, and **Sentry**—see the upstream README for the full stack.

## Documentation

**Canonical docs** (user, admin, self-host, developer, API reference, FAQ) live at **[tux.atl.dev](https://tux.atl.dev)**—not in this Mintlify project.
